Examples
========

Runnable scripts demonstrating pyTelops. Each needs a connected Telops camera.

* ``01_connect_and_grab.py`` - discover, connect, grab one calibrated frame.
* ``02_continuous_live.py`` - continuous acquisition with ``latest=True`` for
  a lag-free live display.
* ``03_buffer_record.py`` - onboard high-speed buffer recording and download.
* ``04_calibration_load.py`` - load a calibration collection by lens and
  target temperature.
* ``05_external_trigger.py`` - buffer recording started by an external BNC
  trigger.
